# PhotoEditor
Projects for Computer Graphics at Warsaw University of Technology.

![Screenshot](https://raw.githubusercontent.com/davidmigloz/computer-graphics/master/src/main/resources/screenshot.gif)

### Features
- Histogram
- Functional filters:
  + Inversion
  + Brightness
  + Contrast 
  + Gamma
  + Curves
- Matrix filters:
  + Blur
  + Gaussian blur
  + Sharpen
  + Edges
  + Emboss
  + Personalized
